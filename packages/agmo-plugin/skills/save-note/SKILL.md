# Agmo Save Note

Use this when the result should survive outside transient chat.

## Main behavior

1. prepare the note through `agmo-wisdom`
2. keep the note focused on one durable artifact
3. prefer concise summaries over transcript dumps
4. preserve title, tags, and nearby project linkage
