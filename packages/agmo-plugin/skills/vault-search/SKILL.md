# Agmo Vault Search

Use this before planning or execution when prior durable context matters.

## Main behavior

1. delegate retrieval to `agmo-wisdom`
2. search the current project note family first
3. return only the strongest matching notes
4. distinguish canonical notes from weak duplicates
